【python GUI】
(方法1)
python calculator_app.py

(方法2) 產生exe執行檔
1. 安裝 PyInstaller：
pip install pyinstaller

2. 將程式轉換為 EXE：
打開命令提示字元，導航到 Python 腳本的目錄，然後運行以下命令：
pyinstaller --onefile --windowed calculator_app.py

3. 找到生成的 EXE：
完成後，您可以在 dist 目錄中找到生成的 EXE 檔案。

4. 測試 EXE：
在 dist 目錄中雙擊生成的 EXE 檔案以運行應用程序，確認其正常工作。
